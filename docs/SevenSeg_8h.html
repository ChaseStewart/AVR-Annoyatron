Update one of the indices in the display buffer with a new value to be written to the SevenSeg display next time writeSevenSeg() is called.

Parameters
indexIndex of the display char to update; 0-1 are the chars before the colon, 2 is the colon itself, 3-4 are the chars after the colon
valueThe value to set the provided index to- values are 0-9 or SEVENSEG_ALL or SEVENSEG_NONE
Returns
None

display_buffer[2] controls the colon. This will set colon unless value is one of {0, SEVENSEG_NONE}
